# История изменений

Все значимые изменения в **Antigravity SubAgents** документируются здесь.

## [0.0.7] - 2025-12-08

### Изменено
- **Маршрутизация SubAgent**: Реализован строгий протокол "Capability Scan". Предотвращает выполнение ручной работы Оркестратором, если существует подходящий SubAgent.

## [0.0.6] - 2025-12-08

### Изменено
- **Иконки**: Обновлены иконки расширения (логотип).

## [0.0.5] - 2025-12-08

### Документация
- **Предварительные требования**: Добавлен раздел с описанием требования установленных и авторизованных **Codex CLI** и **Claude Code CLI**.

## [0.0.4] - 2025-12-08

### Документация
- **Уточнение**: Доработан `README` для корректной идентификации **Antigravity** как IDE и Main Agents как **Оркестраторов**.
- **Возможности**: Выделено "Фокусированное выполнение" как ключевое преимущество — Sub-Agents выполняются только с узкими инструкциями.

## [0.0.3] - 2025-12-08

### Документация
- **README**: Полностью переписан для отражения новой архитектуры **Antigravity Orchestrator**.
- **Перевод**: Обновлён `README_ru.md` с точной документацией на русском языке.

## [0.0.2] - 2025-12-08

### Исправлено
- **Права песочницы**: Sub-Agents теперь запускаются с `danger-full-access` для возможности создания файлов в корне проекта.
- **Манифест**: Обновлена конфигурация агента `translator` по умолчанию.

## [0.0.1] - 2025-12-08

### Добавлено
- **Первый релиз** Antigravity SubAgents!
- **Поддержка оркестратора**: Превращает Antigravity (Gemini CLI) в мощный Оркестратор.
- **Авто-маршрутизация**: Автоматически внедряет инструкции маршрутизации в `~/.gemini/GEMINI.md`.
- **Генерация воркфлоу**: Создаёт Slash Commands для Antigravity в `.agent/workflows/` (Проект) и `~/.gemini/antigravity/global_workflows/` (Глобально).
- **Поддержка Sub-Agent**: Создание и управление Sub-Agents, выполняемыми **Codex CLI** или **Claude Code CLI**.
- **UI**: VS Code Webview для визуального управления агентами (Создание, Редактирование, Деплой, Библиотека).
