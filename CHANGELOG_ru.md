# Список изменений

Все важные изменения в **Antigravity SubAgents** документируются здесь.

## [0.0.7] - 2025-12-08

### Изменено
- **Маршрутизация Суб-Агентов**: Внедрен строгий протокол "Capability Scan". Предотвращает выполнение ручной работы Оркестратором, если существует подходящий Суб-Агент.

## [0.0.6] - 2025-12-08

### Изменено
- **Иконки**: Обновлены иконки расширения (логотип).

## [0.0.5] - 2025-12-08

### Документация
- **Предварительные требования**: Добавлен раздел, описывающий требования к установке и аутентификации **Codex CLI** и **Claude Code CLI**.

## [0.0.4] - 2025-12-08

### Документация
- **Уточнение**: `README` уточнен для правильного позиционирования **Antigravity** как IDE и Основных Агентов как **Оркестраторов**.
- **Функции**: Выделено "Фокусное выполнение" как ключевое преимущество — Суб-Агенты запускаются только с узкими инструкциями.

## [0.0.3] - 2025-12-08

### Документация
- **README**: Полностью переписан для отражения новой архитектуры **Оркестратора Antigravity**.
- **Перевод**: Обновлен `README_ru.md` с точной документацией на русском языке.

## [0.0.2] - 2025-12-08

### Исправлено
- **Права доступа в песочнице**: Суб-Агенты теперь запускаются с `danger-full-access`, чтобы разрешить создание файлов в корне проекта.
- **Манифест**: Обновлена конфигурация агента `translator` по умолчанию.

## [0.0.1] - 2025-12-08

### Добавлено
- **Первый релиз** Antigravity SubAgents!
- **Поддержка Оркестратора**: Превращает Antigravity (Gemini CLI) в мощный Оркестратор.
- **Авто-маршрутизация**: Автоматически внедряет инструкции маршрутизации в `~/.gemini/GEMINI.md`.
- **Генерация рабочих процессов**: Создает Slash-команды для Antigravity в `.agent/workflows/` (Проект) и `~/.gemini/antigravity/global_workflows/` (Глобально).
- **Поддержка Суб-Агентов**: Создание и управление Суб-Агентами, выполняемыми **Codex CLI** или **Claude Code CLI**.
- **UI**: VS Code Webview для визуального управления агентами (Создание, Редактирование, Развертывание, Библиотека).
