# История изменений

Все заметные изменения в **Antigravity SubAgents** задокументированы здесь.

## [0.0.1] - 2025-12-08

### Добавлено
- **Первый релиз** Antigravity SubAgents.
- **Поддержка оркестратора**: превращает Antigravity (Gemini CLI) в мощный оркестратор.
- **Автоматическая маршрутизация**: автоматически добавляет инструкции маршрутизации в `~/.gemini/GEMINI.md`.
- **Генерация рабочих процессов**: создаёт Slash-команды для Antigravity в `.agent/workflows/` (проект) и `~/.gemini/antigravity/global_workflows/` (глобально).
- **Поддержка субагентов**: создание и управление субагентами, выполняемыми через **Codex CLI** или **Claude Code CLI**.
- **Интерфейс**: веб-представление VS Code для визуального управления агентами (создание, редактирование, деплой, библиотека).
