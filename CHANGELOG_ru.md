# История изменений

Все значимые изменения проекта **Antigravity SubAgents** документируются в этом файле.

## [0.0.3] - 2025-12-08

### Документация
- **README**: Полностью переписан для соответствия новой архитектуре **Antigravity Orchestrator**.
- **Перевод**: Обновлен `README_ru.md` с актуальной документацией на русском.

## [0.0.2] - 2025-12-08

### Исправлено
- **Права Sandbox**: Sub-Agent'ы теперь запускаются с правами `danger-full-access`, что позволяет им создавать файлы в корне проекта.
- **Манифест**: Обновлена конфигурация агента `translator` по умолчанию.

## [0.0.1] - 2025-12-08

### Добавлено
- **Первый релиз** Antigravity SubAgents!
- **Поддержка Оркестрации**: Превращает Antigravity (Gemini CLI) в мощного Оркестратора.
- **Авто-маршрутизация**: Автоматически внедряет инструкции маршрутизации в `~/.gemini/GEMINI.md`.
- **Генерация Workflows**: Создает Slash-команды для Antigravity в `.agent/workflows/` (Проект) и `~/.gemini/antigravity/global_workflows/` (Глобально).
- **Поддержка Суб-агентов**: Создание и управление суб-агентами, выполняемыми через **Codex CLI** или **Claude Code CLI**.
- **UI**: Интерфейс VS Code Webview для визуального управления агентами (Создание, Редактирование, Развертывание, Библиотека).
