# Журнал изменений

Все заметные изменения в **Antigravity SubAgents** документируются здесь.

## [0.0.8] - 2025-12-08

### Исправлено
- **Codex CLI**: Удален невалидный флаг `-a on-request` из генерируемых команд запуска. Это исправляет ошибку "unexpected argument" при запуске субагентов.

### Изменено
- **Протокол маршрутизации**: В инструкции `GEMINI.md` добавлен новый **ПРОТОКОЛ ВЗАИМОДЕЙСТВИЯ** (INTERACTION PROTOCOL).
- **Команда Resume**: Описано, как обрабатывать вопросы от субагентов с помощью `commands.resume`.

## [0.0.7] - 2025-12-08

### Изменено
- **Маршрутизация SubAgent**: Реализован строгий протокол «Сканирование возможностей» (Capability Scan), который предотвращает выполнение Оркестратором ручной работы, если существует подходящий SubAgent.

## [0.0.6] - 2025-12-08

### Изменено
- **Иконки**: Обновлены иконки расширения (логотип).

## [0.0.5] - 2025-12-08

### Документация
- **Предварительные требования**: Добавлен раздел, описывающий необходимость установленного и аутентифицированного **Codex CLI** и **Claude Code CLI**.

## [0.0.4] - 2025-12-08

### Документация
- **Уточнение**: Обновлён `README` с корректным указанием **Antigravity** как IDE и Главных Агентов как **Оркестраторов**.
- **Возможности**: Выделена «Фокусированная обработка» как ключевое преимущество — SubAgents запускаются только с узкими (конкретными) инструкциями.

## [0.0.3] - 2025-12-08

### Документация
- **README**: Полностью переписан с учётом новой архитектуры **Antigravity Orchestrator**.
- **Перевод**: Обновлён `README_ru.md` с актуальной русскоязычной документацией.

## [0.0.2] - 2025-12-08

### Исправлено
- **Права песочницы**: SubAgents теперь запускаются с `danger-full-access`, что позволяет создавать файлы в корне проекта.
- **Манифест**: Обновлена конфигурация агента `translator` по умолчанию.

## [0.0.1] - 2025-12-08

### Добавлено
- **Первый релиз** Antigravity SubAgents!
- **Поддержка Оркестратора**: Превращает Antigravity (Gemini CLI) в мощный Оркестратор.
- **Авто-маршрутизация**: Автоматически внедряет инструкции маршрутизации в `~/.gemini/GEMINI.md`.
- **Генерация рабочих процессов**: Создаёт Slash-команды для Antigravity в `.agent/workflows/` (проект) и `~/.gemini/antigravity/global_workflows/` (глобально).
- **Поддержка SubAgent**: Создание и управление SubAgents, исполняемых через **Codex CLI** или **Claude Code CLI**.
- **Интерфейс**: Веб-представление VS Code для визуального управления агентами (Создание, Редактирование, Деплой, Библиотека).
